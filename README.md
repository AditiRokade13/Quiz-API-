📚 Quiz Game using JavaScript & API

This project is a web-based quiz game built with HTML, CSS, and JavaScript, where users can test their knowledge by answering multiple-choice questions.
The original version of the quiz contained a fixed set of questions. To make the game more dynamic and engaging, I enhanced it by integrating a public API that fetches new questions every time the user plays. This ensures that the quiz experience stays fresh and users can continue playing without repeating the same questions.

✨ Features
Interactive multiple-choice quiz interface
Questions fetched dynamically using an API
Score tracking system
Clean and responsive UI
Prevents repetition and increases replay value

🛠️ Technologies Used
HTML – Structure of the application
CSS – Styling and layout
JavaScript – Game logic and API integration
API – Fetching quiz questions dynamically

🎯 Goal of the Project
The goal of this project was to practice JavaScript concepts such as DOM manipulation, asynchronous programming (fetch API), and handling dynamic data while building a fun and interactive web application.
